# Analyse exploratoire des données Netflix

> Exploratory Data Analysis / Media Analytics

## Vue d'ensemble

Analyse exploratoire d'un dataset Netflix pour comprendre la composition du catalogue : types de contenus, pays, années de sortie, genres et tendances d'ajout à la plateforme.

## Objectifs du projet

- Explorer la structure du catalogue Netflix.
- Identifier les tendances par type, pays, année et genre.
- Produire des visualisations synthétiques.
- Mettre en avant des insights exploitables pour une lecture produit/contenu.

## Démarche

- Chargement et inspection des données.
- Nettoyage des valeurs manquantes et formats.
- Analyse des distributions et catégories dominantes.
- Visualisations et interprétation des tendances.

## Stack technique

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- EDA
